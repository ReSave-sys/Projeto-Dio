# Dashboard de Vendas - Excel

Projeto desenvolvido como parte de um desafio do curso da DIO.  
O objetivo é criar um dashboard de vendas no Excel para análise visual dos dados.

## Dados
Dados fictícios de vendas fornecidos pela DIO, utilizados apenas para fins educacionais.

## Ferramentas

- Tabelas Dinâmicas
- Gráficos
